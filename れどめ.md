# CORSエラー
かんたんに言えば、フロントエンドとバックエンドのポート間での許可ができていない状態。
これを直すのが、フロントとバックの競合の練習になる

backend 内部の main.ts に CORS 設定を許可するコマンドを入れることで直せる。
